# Codex Debug Prompt

## 使用场景
当代码报错、行为异常，或者我“知道不对但说不清楚哪里不对”时使用。

---

## Prompt 模板

```text
你是我的资深工程师搭档。

这是当前代码和问题描述：
[在这里粘贴代码 / 报错 / 现象]

请你：
1. 先判断问题可能出现在哪些层面（逻辑 / 边界 / API 使用 / 环境）
2. 给出最可能的原因排序
3. 提供最小修改建议，而不是直接重写全部代码
4. 如果有多种方案，请说明取舍理由
